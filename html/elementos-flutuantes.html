Elementos Flutuantes (Float)

Antes da criação do Flexbox e do CSS Grid, o float era a principal técnica utilizada para construir layouts com colunas. Atualmente ele ainda é importante, principalmente para compreender códigos antigos e posicionar imagens ao lado de textos.

1. Float Left

Left

A caixa azul está utilizando float: left. Observe que o texto passa a ocupar automaticamente o espaço disponível ao lado da caixa.

2. Float Right

Right

Agora o elemento está utilizando float: right. Todo o conteúdo tenta preencher o espaço livre à esquerda.

3. Float None

None

O valor none cancela qualquer flutuação. O elemento volta ao fluxo normal da página.

5. Exemplo Prático

Foto

Um dos usos mais comuns do float é posicionar uma imagem ao lado de um texto, permitindo que o conteúdo envolva a imagem de forma semelhante ao que ocorre em jornais, revistas e blogs.

Durante muitos anos essa foi uma das aplicações mais frequentes do float.

6. Observação Importante

Atualmente, para criar layouts completos, recomenda-se utilizar Flexbox ou CSS Grid. O float continua existindo, porém sua finalidade principal hoje é posicionar pequenos elementos, como imagens ao lado de textos.

Resumo

  • float: left; → Flutua para a esquerda.
  • float: right; → Flutua para a direita.
  • float: none; → Remove a flutuação.
  • clear: left; → Limpa elementos à esquerda.
  • clear: right; → Limpa elementos à direita.
  • clear: both; → Limpa ambos os lados.
  • .clearfix → Corrige o colapso do elemento pai quando todos os filhos utilizam float.
